Get started
=======================================

Clone
------

.. code-block:: bash

  git clone https://github.com/linagora/twake-doc-fr
  
  cd twake-doc-fr

  python3 pip install requirement.txt
  
  # edit files inside docs/source/


Build locally
------

.. code-block:: bash
  
  cd docs

  make -b html

  python3 -m sphinx source build

  # open docs/build/html/index.html
